# aalo [আলো](https://translate.google.com/?sl=bn&tl=en&text=%E0%A6%86%E0%A6%B2%E0%A7%8B&op=translate)

```text
in bengali, aalo means "light" (i.e. photons), not to be confused with haalka !
```

[aalo](https://github.com/databasedav/aalo) aka bevy-inspector-haalka is a [haalka](https://github.com/databasedav/haalka) port of [bevy-inspector-egui](https://github.com/jakobhellermann/bevy-inspector-egui).
